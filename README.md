## Set Up
1. Go to [this](https://github.com/Maia-Cochran/chefYeezy) repository and clone it. 
2. `cd` into the directory.
3. Run `npm i` in your terminal
4. Run `npm run start-dev`
6. Go to http://localhost:3000/ to view the site.
